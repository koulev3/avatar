# Avatar Campaign Workspace

This folder mirrors the `corona-eclipsa` project layout, but tuned for an Avatar Legends campaign centered on the Tea Incident.

## Layout
- `GAME.md` - campaign defaults, tone, and planning rules
- `continuity/` - persistent canon (PC and world truths)
- `plans/` - session-by-session prep docs for table use
- `references/` - reusable tools and prompt packs
- `player-json/` - optional structured player payloads for automation

## Start Here
1. Review `GAME.md`.
2. Fill in PC continuity docs (beginning with `continuity/toren-character-summary.md`).
3. Duplicate the newest plan in `plans/` and update for the next session.
4. Run agent prompts from `references/agent-prompts.md` in the suggested order.
